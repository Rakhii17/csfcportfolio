# üìñ Project Description

This repository contains the complete source code of my portfolio website.
The design features a **soft pink theme** with shades of red, rounded elements, smooth animations, and simple typography (Poppins).

The website is created as a **single-page portfolio**, where the navigation menu smoothly scrolls to different sections such as:

* About
* Skills
* Projects

---

# ‚ú® Key Features

1. **Responsive Design**
   Layout built using CSS Grid and Flexbox, making the site look good on mobile, tablet, and desktop.

2. **CSS-Only Project Image Slider**
   Each project card uses a custom `@keyframes` animation that gently transitions between screenshots automatically without JavaScript.

3. **Sticky Header Navigation**
   The header stays visible at the top while scrolling for easier navigation.

4. **Smooth Page Scrolling**
   All anchor links and the "Back to Top" button use CSS‚Äôs native smooth scrolling.

5. **Modern UI Elements**

   * Hover animations on skill cards
   * Circular profile image with border and shadow
   * Section cards with soft shadows
   * Gradient headers and soft theme colors

6. **CSS Variables**
   Custom `:root` variables make color and theme customization easy.

# üöÄ Installation and Usage

No installations or build steps (like Node.js or Python) are required since this is a static website.

## Requirements

* Any modern browser (Chrome, Safari, Firefox, Edge, etc.)
* Code editor (VS Code recommended) if you want to modify the site

## Steps

1. **Download or Clone the Project**

   ```bash
   git clone <https://github.com/Rakhii17/portfolio-webpage.git>
   ```

   Or download the ZIP file and extract it.

2. **Add Images**
   Place all your project and profile images into the project directory and ensure filenames match the ones in your HTML.

3. **Open the Site**

   * Double-click `index.html`
   * The website will open in your browser
   * (Optional) Use "Live Server" in VS Code for auto-refresh

# üé® Customization

### Change Text

Open `index.html` and edit:

* Name
* Introduction
* About Me paragraph
* Skill names
* Project descriptions

### Change Colors

All colors are defined inside the `:root` section of `style.css`:

```css
:root {
  --soft-red-bg:#fff0f1;
  --soft-red-mid:#ffd7db;
  --soft-red-accent:#ff9aa9;
  --soft-red-dark:#d94b61;
}
```

Update them to any color palette you want.
